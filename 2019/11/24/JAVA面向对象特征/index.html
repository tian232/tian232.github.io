JAVA面向对象特征：封装，继承，多态

封装：通过方法将对象的数据和实现细节保护起来

继承：单继承–类的层次结构为树型结构 多继承–网状结构

多态：运行是多态，编译时多态

重载实现编译时多态
类之间继承性，方法从写以及晚联编技术实现运行时多态

类的定义

类的基本结构

1.类有两种基本成分：成员变量和成员方法
2.类的定义分为：类声明和类体

成员变量

（1）方法声明

当一个方法不需要返回值时，返回类型为void
（2）方法体
方法体是对方法的实现
在方法体中使用return退出该方法格式；
带返回值和不带返回值
如果带返回值，要在return后面加上相应的值和表达式

方法重载

（1）方法发参数表必须不同，包括参数的类型或个数，以此区分不同方法体
（2）方法发返回类型、修饰符可以相同也可不同

this 指向当前对象自身

构造方法

特点

（1）构造方法的名称必须与类名相同
（2）构造方法不能有返回值
（3）用户不能直接调用构造方法，必须通过关键字new自动调用它
